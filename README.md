# DA-Backend
Gemeinsames Spring Boot Backend für Diplomarbeitsprojekte

## Voraussetzungen

Getestet mit:

- SDK Eclipse Temurin (AdoptOpenJDK HotSpot) 21.0.4 und Amazon Corretto 17.0.12
- Gradle 8.10.1

## Enthaltene Spring Abhändigkeiten

- Spring Boot DevTools
- Lombok
- Spring Web
- Spring Data JPA
- H2 Database
- Validation
